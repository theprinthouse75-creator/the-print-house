THE PRINT HOUSE PREMIUM WEBSITE

Upload instructions:
1. Extract this ZIP.
2. Upload index.html, style.css, script.js and assets folder to GitHub repository.
3. Go to repository Settings > Pages.
4. Source: Deploy from branch.
5. Branch: main, Folder: /root.
6. Save. Website will be live in a few minutes.

Your live URL will be:
https://theprinthouse75-creator.github.io/the-print-house

To add real product photos:
- Put photos inside assets/gallery
- Replace gallery sample boxes in index.html with image tags.
